Social

Social

Not yet supported in the SDK!

The SDK doesn't currently contain any helper annotations, classes or interfaces to ease integration of social, but it's on our roadmap. Let us know if it's a blocker for you and we'll see how we can help.

It's still possible to use all the raw API objects and write your own social controllers though, however we don't yet have any examples of this.
